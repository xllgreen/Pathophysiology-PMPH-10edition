# 遗传性凝血因子缺陷出血机制分析

## 适用场景
- 患者有明确家族性出血史
- 出现不明原因的关节/肌肉出血、术后出血不止、皮肤瘀斑、鼻衄、月经过多或拔牙后出血
- 实验室检查提示内源性凝血途径异常（如APTT延长）

## 执行步骤

1. **判断可能的疾病类型**：
   - 若以深部组织或关节出血为主 → 考虑血友病A或B
   - 若以皮肤黏膜出血为主 → 考虑血管性血友病（vWD）

2. **根据疑似类型分析分子机制**：
   - **血友病A（FVIII缺乏）**：
     - F8基因突变 → FVIII合成减少或功能异常 → 内源性凝血途径障碍 → 出血倾向
   - **血友病B（FIX缺乏）**：
     - F9基因突变 → FIX缺乏或活性降低 → 内源性凝血途径障碍 → 出血倾向
   - **血管性血友病（vWD）**：
     - VWF基因突变 → vWF缺乏或功能异常，导致双重机制：
       a) vWF-FVIII复合体不稳定 → FVIII半衰期缩短 → FVIII水平下降
       b) 血小板无法有效黏附于受损血管内皮 → 初期止血失败

3. **输出临床指导建议**：
   - 推荐进行F8、F9或VWF基因检测以确诊
   - 根据缺陷因子选择替代治疗（如FVIII浓缩物用于血友病A，FIX浓缩物用于血友病B，DDAVP对部分1型vWD有效）
   - 提供遗传咨询，评估家系携带状态与再发风险

> 注意：本技能不适用于获得性凝血因子缺乏（如肝病、维生素K缺乏、抗凝药物使用等）。
