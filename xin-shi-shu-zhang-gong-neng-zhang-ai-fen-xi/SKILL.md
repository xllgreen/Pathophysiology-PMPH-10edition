# 心室舒张功能障碍分析

## 何时使用本技能
- 患者为高血压心脏病或心力衰竭患者
- 存在心肌缺血、肥大、纤维化或衰老等病理基础
- 出现心室充盈受限、舒张末压升高、呼吸困难等症状
- 超声心动图提示舒张功能异常（如E/A比异常、e'速度降低）
- **不适用于**正常舒张功能的评估

## 执行步骤

### 1. 判断是否满足触发条件
IF 患者存在心室充盈受限或左室舒张末压（LVEDP）升高 → 启动本分析流程。

### 2. 分析主动性舒张功能障碍
- **核心特征**：能量依赖过程受损，常早于收缩功能障碍出现
- **检查要点**：
  - 是否存在心肌缺血或缺氧（导致ATP供应不足）
  - 是否表现为心室舒张迟缓、不完全
- **机制确认**：
  - 肌质网/肌膜钙泵活性降低 → 胞质Ca²⁺清除延迟
  - ATP缺乏 + Ca²⁺-肌钙蛋白亲和力增加 → 肌球-肌动蛋白复合体解离困难

### 3. 分析被动性舒张功能障碍
- **核心特征**：心室顺应性（dV/dp）降低，僵硬度（dp/dV）增加
- **检查要点**：
  - 是否存在心室壁增厚（如向心性肥大）、心肌纤维化或间质增生
  - 舒张末期压力-容积（P-V）曲线是否左移
  - 是否需更高充盈压维持正常充盈量
- **临床表现确认**：
  - LVEDP升高（>12 mmHg）
  - 肺静脉压升高 → 肺淤血/肺水肿
  - 每搏输出量减少（因舒张末容积减少）

### 4. 综合判断与治疗导向
- 若以**主动性障碍为主** → 优先改善心肌供能（如抗缺血治疗）
- 若以**被动性障碍为主** → 优先抗纤维化、控制血压、减慢心率
- 即使收缩功能正常，若存在明显被动障碍，仍可诊断为舒张性心力衰竭

> 提示：结合超声心动图参数（E/A比、e'速度、E/e'比值）进行客观评估。
