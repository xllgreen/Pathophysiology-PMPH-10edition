# 体循环淤血临床评估

## 何时使用
- 患者存在右心衰竭或全心衰竭病史
- 出现以下任一体征：下肢水肿、颈静脉充盈/怒张、肝肿大伴压痛、食欲缺乏、恶心、腹泻、胸腔或腹腔积液
- 需鉴别是否为心源性水肿而非其他原因（如肾性、肝性水肿）

## 执行步骤
1. **确认适用对象**：仅适用于右心衰竭或全心衰竭患者；排除单纯左心衰竭且无体循环受累者。
2. **识别关键体征**：
   - 观察颈静脉是否充盈或怒张
   - 检查肝脏是否肿大、有无压痛
   - 进行肝颈静脉反流征检查（按压肝脏后颈静脉异常充盈为阳性）
   - 评估水肿分布（通常始于双下肢，严重时波及全身）
3. **分析机制**：
   - 右室舒张末期压力增高 → 阻碍腔静脉回流 → 静脉系统充盈过度
   - 静脉压升高叠加容量血管收缩 → 毛细血管静水压增高 → 水肿形成
   - 继发因素包括：肾灌注减少致水钠潴留、胃肠道淤血致吸收障碍、肝功能受损致低蛋白血症、淋巴回流障碍
4. **输出判断**：若上述体征组合存在，应高度怀疑体循环淤血，并评估其严重程度以指导利尿、强心等治疗策略。

> **注意**：肝颈静脉反流征阳性具有重要鉴别价值，可帮助区分心源性与其他原因引起的颈静脉怒张。
