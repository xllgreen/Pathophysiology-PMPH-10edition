# 受体数量与亲和力的调节机制

## 何时使用
- 解释持续高浓度配体（如激素、神经递质）引起的受体下调或减敏现象
- 分析配体缺乏导致的受体上调或增敏机制
- 评估磷酸化、内吞、寡聚化等对受体功能的影响
- 判断贮备受体（spare receptors）在药效中的作用

## 执行步骤
1. **判断调节方向**：
   - 若配体长期高浓度 → 触发**受体下调**（内吞+溶酶体降解）或**减敏**（亲和力下降）
   - 若配体长期缺乏 → 可能触发**受体上调**或**增敏**（反应性增强）
2. **分析数量调节机制**：
   - **内吞与降解**：配体-受体复合物经网格蛋白包被小窝内化，部分受体被降解
   - **循环与滞留**：部分受体循环回膜；部分滞留胞内无法转运至膜
3. **分析亲和力调节机制**：
   - **磷酸化修饰**：PKA/PKC磷酸化受体 → 降低亲和力；磷酸酶作用可逆转
   - **构象变化**：变构或寡聚化改变结合位点
   - **信号脱耦联**：受体结合配体但无法激活下游效应器
4. **评估生理/病理意义**：
   - 生理性：缓冲配体波动，维持稳态
   - 病理性：如心衰时β受体下调、哮喘时β受体减敏 → 影响药物疗效
5. **输出结果**：提示细胞敏感性发生适应性或病理性改变，可能需调整用药策略。

> 注意：仅适用于持续性配体刺激（非瞬时刺激）且细胞具备内吞能力；不适用于一次性信号事件。
