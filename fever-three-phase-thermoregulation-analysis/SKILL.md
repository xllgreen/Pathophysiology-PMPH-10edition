# 发热三时相体温调节分析

## 适用条件
- 患者存在明确发热（调定点上移所致）
- 已排除非发热性体温升高（如中暑、剧烈运动后过热）
- 观察到典型发热过程中的体温变化与伴随症状

## 执行步骤

### 1. 判断患者所处发热时相
根据体温趋势、主观感受及体征，确定属于以下哪一阶段：
- **体温上升期**：畏寒、寒战、“鸡皮疙瘩”、皮肤苍白发凉
- **高温持续期**：自觉酷热、皮肤潮红、口唇干燥、无寒战
- **体温下降期**：大量出汗、皮肤湿润、体温逐渐或迅速回落

### 2. 分析各时相的体温调节机制
- **体温上升期**：
  - 调定点上移（>37℃），原体温被视为“冷刺激”
  - 交感神经兴奋 → 皮肤血管收缩 → 散热减少
  - 寒战中枢激活 → 骨骼肌节律性收缩 → 产热增加4~5倍
  - 棕色脂肪分解氧化 → 非寒战产热增加
  - 热代谢特点：产热 > 散热

- **高温持续期**：
  - 体温达新调定点水平，寒战停止
  - 产热与散热在高水平平衡
  - 皮肤血管扩张、血流量增加、蒸发散热增强
  - 持续时间因病因而异（疟疾数小时，伤寒可超1周）

- **体温下降期**：
  - 致热原及发热介质清除，调定点恢复至37℃
  - 血温 > 调定点 → 热敏神经元兴奋 → 交感紧张性降低
  - 皮肤血管进一步扩张，大量出汗 → 散热显著增加
  - 产热减少，体温下降（可为骤退或渐退）

### 3. 输出结果
根据所处时相，明确：
- 当前热代谢状态（产热 vs 散热）
- 典型临床表现
- 护理重点（如保暖、补液、监测脱水风险等）

> **注意**：本技能不适用于无调定点改变的体温波动或非感染/炎症性体温升高。
