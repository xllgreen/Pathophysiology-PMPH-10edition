# II型呼吸衰竭控制性氧疗

## 适用条件
- 患者已确诊为II型呼吸衰竭（即同时存在低氧血症和高碳酸血症）
- 可持续监测动脉血气（PaO₂和PaCO₂）
- **不适用于** I型呼吸衰竭或无高碳酸血症风险的缺氧患者

## 执行步骤
1. **确认诊断**：核实动脉血气结果符合II型呼吸衰竭标准（PaO₂ < 60 mmHg 且 PaCO₂ > 50 mmHg）。
2. **设定氧疗目标**：将PaO₂维持在50–60 mmHg，避免完全纠正缺氧。
3. **控制吸氧浓度**：使用低流量鼻导管（1–2 L/min），确保吸入氧浓度（FiO₂）不超过30%。
4. **理解生理机制**：II型呼吸衰竭患者依赖低氧刺激驱动呼吸；若PaO₂过高，会抑制呼吸中枢，加重CO₂潴留，可能导致呼吸性酸中毒恶化甚至昏迷。
5. **持续监测**：定期复查动脉血气，确保PaO₂在目标范围内且PaCO₂未显著升高。
6. **对比参考**：I型呼吸衰竭（仅缺氧、无CO₂潴留）可给予较高浓度氧（≤50%），无需严格限制FiO₂。

## 关键参数
- **目标PaO₂**：50–60 mmHg
- **最大FiO₂**：0.30（30%）

执行本技能后，应记录氧疗方案、血气变化及患者反应，必要时调整治疗策略。
