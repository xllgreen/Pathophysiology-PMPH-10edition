# 高血糖所致物质代谢紊乱特征分析

## 适用条件
- 胰岛素绝对或相对缺乏
- 高血糖状态持续存在
- 不适用于短期应激性高血糖或胰岛素敏感个体

## 执行步骤

1. **评估糖代谢紊乱**：
   - 检查肝、骨骼肌、脂肪组织对葡萄糖的摄取和利用是否减少；
   - 判断肝糖原分解是否增强，进一步升高血糖。

2. **评估脂代谢紊乱**：
   - 检查脂肪组织从血液摄取甘油三酯是否减少，脂肪合成是否降低；
   - 评估脂蛋白酯酶（LPL）活性是否下降，导致血中游离脂肪酸（FFA）和甘油三酯（TG）浓度升高。

3. **评估蛋白质代谢紊乱**：
   - 判断蛋白质合成是否减少、分解是否加速；
   - 确认是否存在负氮平衡（摄入氮 < 排出氮）；
   - 观察是否出现体重减轻等临床表现。

4. **综合判断**：
   - 若同时存在上述三类代谢异常，并伴有“三多一少”症状，则支持高血糖所致物质代谢紊乱的诊断。

5. **根本原因确认**：
   - 明确是否由胰岛素质或量缺陷和/或胰岛素生物学效应降低引起。

## 输出结果
生成评估结论，明确患者是否存在负氮平衡、高甘油三酯血症及持续高血糖状态。
