## プロジェクト概要

本プロジェクトは、酸塩基平衡障害、ショックと多臓器不全、水電解質代謝異常、炎症と免疫応答、細胞シグナル伝達、発癌メカニズム、老化関連病理プロセス、呼吸・循環機能障害、神経精神障害、腎臓病理生理学、内分泌代謝疾患など、病理生理学の **145の中核的能力**を体系的に統合しています。

**対象者**：医学生、病理生理学教育者、臨床医、基礎医学研究者

**参考教科書**：人民衛生出版社『病理生理学』第10版

**⚠️ リスク ⚠️**：このスキルセットは疾患メカニズム分析、診断推論、治療原則を網羅しており、独立した臨床判断の根拠として誤用される可能性があります。

対策：出力は教育的または臨床医によるレビューのための参考資料としてのみ使用し、現在の公式ガイドライン、地域プロトコル、および資格を持つ専門家の意見に照らして推奨事項を検証してください。
