# RAS基因突变致瘤机制分析

## 何时使用
- 检测到KRAS、NRAS或HRAS基因发生功能获得性点突变（如G12D、Q61K等）
- 需判断该突变是否为肿瘤驱动事件
- 需评估RAS-MAPK信号通路激活状态以指导靶向治疗选择
- 肿瘤类型属于RAS突变高发谱系（如胰腺癌、结直肠癌、肺癌、黑色素瘤、AML等）

## 执行步骤
1. **确认突变信息**：明确突变基因（KRAS/NRAS/HRAS）、突变位点（如密码子12、13、61）及具体变异类型（如G12D）。
2. **判断RAS蛋白激活状态**：若突变位于关键功能域（如GTPase结构域），则RAS蛋白丧失GTP水解能力，持续处于GTP结合的激活状态。
3. **分析下游通路影响**：持续激活的RAS异常激活RAF-MEK-ERK（MAPK）信号通路，导致细胞增殖、生长、分化和代谢失控。
4. **关联肿瘤类型特异性**：
   - KRAS突变：常见于胰腺癌、结直肠癌、非小细胞肺癌
   - NRAS突变：多见于黑色素瘤、急性髓系白血病（AML）
   - HRAS突变：多见于膀胱癌、甲状腺癌、宫颈癌、头颈癌
5. **排除非突变性激活**：若无RAS基因突变，但MAPK通路仍激活，应考虑上游（如RTK扩增）或下游（如BRAF突变）机制，不适用本技能。
6. **输出结论**：确认RAS突变是否为驱动事件，并评估其对MAPK通路激活的贡献程度，用于指导后续靶向治疗策略（如是否适用MEK抑制剂等）。

> 注意：本技能仅适用于RAS基因本身发生功能获得性突变的情况，不适用于由上游受体酪氨酸激酶（RTK）过度激活等非突变机制引起的RAS通路活化。
