# 心力衰竭按射血分数分类详解

## LVEF计算公式
左心室射血分数（LVEF） = 每搏输出量 / 左心室舒张末容积 × 100%
正常静息状态下LVEF为55%~70%。

## 各亚型特征

### HFrEF（LVEF < 40%）
- **病理特点**：心肌收缩力显著下降，左心室舒张末容积扩大
- **常见病因**：冠心病、扩张型心肌病、心肌炎后遗症
- **治疗药物**：β受体阻滞剂（如美托洛尔）、ACEI（如依那普利）、ARB、ARNI（如沙库巴曲缬沙坦）、MRA、SGLT2抑制剂

### HFmrEF（LVEF 40%–49%）
- **病理特点**：轻度收缩功能障碍合并舒张功能异常
- **临床意义**：部分患者可能从HFrEF治疗中获益，需动态评估
- **研究现状**：DAPA-HF、EMPEROR-Preserved等试验证实SGLT2抑制剂对此类患者有效

### HFpEF（LVEF ≥ 50%）
- **病理特点**：心肌肥厚、纤维化导致舒张功能受损，心室顺应性下降
- **高发人群**：老年女性、长期高血压、糖尿病、肥胖患者
- **治疗重点**：控制血压（目标<130/80 mmHg）、管理容量负荷、改善代谢异常；目前无明确改善预后的神经激素调节药物

## 注意事项
- LVEF测量方法（如Simpson法）影响结果准确性
- 单次LVEF值不足以确诊，需结合症状（如呼吸困难、水肿）、BNP/NT-proBNP水平及其他影像学指标
- 急性失代偿期LVEF可能假性降低，建议病情稳定后复查
