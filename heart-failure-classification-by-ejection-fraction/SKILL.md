# 按射血分数分类心力衰竭

## 何时使用
- 患者已被临床诊断为心力衰竭
- 已完成左心室射血分数（LVEF）测定（通常通过超声心动图）
- 需要根据LVEF值确定心力衰竭亚型以制定治疗方案

> **注意**：不适用于急性心肌梗死早期（此时LVEF可能暂时性下降），且必须结合临床症状综合判断。

## 执行步骤
1. **确认LVEF测定结果**：确保LVEF值来自可靠的影像学检查（如超声心动图）。
2. **按以下标准分类**：
   - **射血分数降低的心力衰竭（HFrEF）**：LVEF < 40%
   - **射血分数中间范围的心力衰竭（HFmrEF）**：LVEF 40%–49%
   - **射血分数保留的心力衰竭（HFpEF）**：LVEF ≥ 50%
3. **关联临床特征与治疗方向**：
   - HFrEF：常伴心腔扩大，可使用β受体阻滞剂、ACEI/ARB、ARNI等改善预后
   - HFpEF：舒张功能异常为主，治疗侧重控制血压、改善心室顺应性
   - HFmrEF：处于过渡状态，需个体化评估
4. **输出分类结果及治疗提示**：明确心力衰竭类型，并说明其对药物选择的影响。
