# 低钙血症的病因与机制分析

## 何时使用
- 血清总钙<2.25 mmol/L（白蛋白正常）
- 需区分维生素D缺乏、PTH异常、肾衰竭等病因

## 执行步骤
1. **评估维生素D代谢障碍**：
   - 摄入/日照不足、肠吸收障碍、肝肾羟化障碍
2. **检查甲状旁腺功能**：
   - PTH缺乏（手术误切）或抵抗（假性甲旁减）
3. **分析慢性肾衰竭机制**：
   - 高磷血症→血钙↓
   - 1,25-(OH)₂D₃生成不足→肠钙吸收↓
4. **考虑其他因素**：
   - 低镁血症→PTH分泌↓
   - 急性胰腺炎→钙皂形成
   - 低白蛋白血症（需校正）
5. **输出病因分类**：识别具体病因类别，指导针对性治疗（如补活性维生素D、PTH替代等）。
