# 周期蛋白与CDK驱动细胞周期的机制

## 适用场景
- 分析肿瘤细胞异常增殖的分子基础
- 评估细胞周期阻滞药物的作用靶点
- 确定细胞所处周期阶段（G₁、S、G₂、M）

## 执行步骤
1. **识别当前周期时相**：根据细胞形态、标志物或实验数据确定阶段。

2. **匹配对应的Cyclin-CDK复合物**：
   - G₁期：CDK4/6 + Cyclin D1/D2/D3
   - G₁/S过渡：CDK2 + Cyclin E
   - S期：CDK2 + Cyclin A
   - G₂/M过渡：CDK1 + Cyclin A
   - M期：CDK1 + Cyclin B
   - 全周期转录调控：CDK7 + Cyclin H

3. **验证前提条件**：
   - 对应Cyclin已表达
   - CDK蛋白稳定存在
   - 无强效CKI（如p21、p27）抑制

4. **确认CDK激活**：Cyclin结合后，CDK构象改变并被CAK磷酸化激活。

5. **执行底物磷酸化**：活化CDK磷酸化丝氨酸/苏氨酸残基，推动周期进程（如Rb蛋白磷酸化促进E2F释放）。

## 输出结果
激活下游信号通路，推动细胞进入下一周期阶段。
