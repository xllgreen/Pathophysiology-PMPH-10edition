# 呼吸性酸中毒代偿调节分析

## 适用场景
- 患者动脉血气分析提示PaCO₂升高、pH下降
- 已明确病程是否超过24小时
- 需排除代谢性酸碱失衡

## 执行步骤

1. **确认前提条件**：
   - PaCO₂ > 45 mmHg
   - pH < 7.35
   - 病程时间已知（<24小时 或 ≥24小时）

2. **判断代偿类型**：
   - 若病程 **<24小时**，视为**急性呼吸性酸中毒**：
     - 肾脏代偿未启动
     - 主要依赖血液缓冲系统（如血红蛋白）和细胞内外离子交换
     - 预期HCO₃⁻变化：PaCO₂每升高10 mmHg，HCO₃⁻仅增加0.7~1 mmol/L
   - 若病程 **≥24小时**（尤其3~5天），视为**慢性呼吸性酸中毒**：
     - 肾脏代偿已激活
     - 肾小管增加H⁺/NH₄⁺排泌和HCO₃⁻重吸收
     - 预期HCO₃⁻变化：PaCO₂每升高10 mmHg，HCO₃⁻增加3.5~4.0 mmol/L

3. **评估代偿状态**：
   - 计算实测HCO₃⁻与预期值的偏差
   - 若实测值接近预期值 → 代偿充分
   - 若实测值显著低于预期值 → 可能合并代谢性酸中毒
   - 若实测值显著高于预期值 → 可能合并代谢性碱中毒

4. **输出结论**：
   - 明确为急性或慢性呼吸性酸中毒
   - 判断是否代偿、部分代偿或失代偿
   - 提示是否存在混合性酸碱失衡可能

> 注意：本技能不适用于代谢性酸碱紊乱。若HCO₃⁻原发降低或升高，应使用其他酸碱分析技能。
