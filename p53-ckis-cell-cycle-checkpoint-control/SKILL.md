# p53与CDK抑制因子介导的细胞周期检查点控制

## 何时使用
- 细胞暴露于DNA损伤因素（如辐射、化学诱变剂）后需评估其命运
- 分析癌前病变细胞是否具备完整的G1检查点功能
- 研究p53野生型状态与CKI表达对细胞周期阻滞的影响
- 判断肿瘤细胞对治疗的敏感性（依赖p53/CKI通路完整性）

## 执行步骤

### 1. 验证前提条件
- 确认细胞类型为增殖中的真核细胞或癌前病变细胞
- 检测p53基因状态：必须为野生型
- 确认细胞处于G1期或G1/S交界处
- 排除终末分化细胞或p53缺失/突变细胞

### 2. 启动p53依赖的检查点程序
- 若检测到染色体DNA损伤，野生型p53被激活并作为转录因子
- p53诱导CDK抑制因子（尤其是p21^Waf1，属Kip家族）表达
- p21结合Cyclin E-CDK2复合物，抑制其激酶活性，导致G1期阻滞

### 3. 整合CKI家族作用机制
- **INK4家族**（如p16^INK4a）：特异性抑制CDK4/6，阻止Cyclin D结合，维持Rb对E2F的抑制
- **Kip家族**（如p21、p27）：广谱抑制CDK2、CDK1等，阻断S期进入或M期进程
- CKI表达可由DNA损伤（经p53）或TGF-β等信号诱导

### 4. 决定细胞命运
- **若DNA损伤可修复**：阻滞期间完成修复，CKI水平下降，细胞恢复周期进程
- **若DNA损伤不可修复**：p53启动凋亡程序，包括：
  - 转录激活Bax、PUMA（线粒体通路）
  - 转录激活Fas、DR5（死亡受体通路）
  - 抑制生存信号通路
  - p53蛋白直接转位至线粒体，模拟BH3-only蛋白功能

### 5. 输出结果
- 细胞命运判定：继续周期 或 启动凋亡清除
- 评估“分子警察”机制是否完整有效

> 注意：在p53突变或缺失细胞中，此通路失效，CKI可能无法被充分诱导，导致检查点崩溃和异常增殖。
