# 近曲小管功能障碍的临床表现识别

## 何时使用
- 血糖正常但尿糖阳性
- 出现氨基酸尿、低分子蛋白尿
- 血HCO₃⁻降低但肾小球滤过率基本正常

## 执行步骤
1. **回顾近曲小管正常功能**：
   - 100%重吸收葡萄糖、氨基酸
   - 重吸收85% HCO₃⁻、60%~70% 钠水
   - 排泄酚红、青霉素等
2. **识别功能障碍表现**：
   - 肾性糖尿（血糖正常+尿糖+）
   - 氨基酸尿
   - 肾小管性酸中毒（HCO₃⁻重吸收↓）
   - 特定药物/造影剂清除延迟
3. **注意矛盾现象**：钠水重吸收减少理论上应导致失钠，但常因代偿表现为水钠潴留。
4. **输出预警**：提示近曲小管功能障碍，需进一步评估肾小管疾病。
