# BRAF V600E突变驱动肿瘤的机制分析

## 何时使用
- 肿瘤分子检测报告提示存在BRAF基因V600E（或V600K等）激活性突变
- 需要解释BRAF突变在特定肿瘤类型中的致病机制
- 为是否采用BRAF抑制剂（如维莫非尼、达拉非尼）或联合MEK抑制剂提供理论依据

## 执行步骤
1. **确认突变状态与肿瘤类型**：核实BRAF_mutation_status（如V600E）和cancer_type（如黑色素瘤、甲状腺乳头状癌、结直肠癌、非小细胞肺癌）。
2. **评估MAPK通路激活状态**：判断MAPK_pathway_activity是否因BRAF突变而持续激活（通常为是）。
3. **解释致病机制**：
   - BRAF是RAS下游的丝氨酸/苏氨酸激酶，属RAF家族，为MAPK通路关键节点。
   - V600E突变使BRAF蛋白构象改变，在无RAS激活情况下仍保持高激酶活性。
   - 持续磷酸化MEK1/2 → 激活ERK → 导致细胞增殖、抗凋亡、代谢重编程等促瘤效应。
4. **结合肿瘤类型判断临床意义**：
   - 黑色素瘤（40–50%）、甲状腺乳头状癌（30–45%）：BRAF V600E常见，单药BRAF抑制剂常有效。
   - 结直肠癌（8–12%）：需警惕EGFR反馈激活，推荐BRAF抑制剂联合EGFR抑制剂（如西妥昔单抗）±MEK抑制剂。
   - 非小细胞肺癌（1–3%）：可考虑BRAF+MEK双靶向治疗。
5. **输出治疗建议**：若确认为激活性BRAF突变，建议使用BRAF抑制剂单药（适用于黑色素瘤等）或联合方案（如结直肠癌需联合EGFR抑制）。

## 注意事项
- 本技能不适用于BRAF失活突变或非激活性变异。
- 在结直肠癌中单独使用BRAF抑制剂可能因EGFR反馈激活而疗效有限，必须考虑联合策略。
