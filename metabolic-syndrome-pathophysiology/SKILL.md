# 代谢综合征的核心代谢异常与进展机制

## 何时使用
- 患者具备≥3项代谢综合征诊断标准
- 出现高尿酸血症伴胰岛素抵抗但无原发痛风
- 需解释为何代谢综合征患者易患心血管疾病

## 执行步骤

### 1. 核查五大核心异常
确认是否存在以下至少三项：
- 空腹血糖 ≥6.1 mmol/L 或糖耐量受损
- 甘油三酯 ≥1.7 mmol/L
- HDL-C <1.0 mmol/L（男）或 <1.3 mmol/L（女）
- 血压 ≥130/85 mmHg
- 腰围超标（男≥90 cm，女≥85 cm，亚洲标准）

### 2. 追溯共同病理基础
- **胰岛素抵抗**是核心驱动力：
  - 靶器官（肌肉、肝、脂肪）对胰岛素敏感性↓
  - 葡萄糖摄取与利用障碍 → 血糖↑
  - 胰岛β细胞代偿性高分泌 → **高胰岛素血症**

### 3. 分析并发症链条
- 高胰岛素血症 → 肾脏尿酸重吸收↑ → **尿酸清除率↓** → 高尿酸血症
- 尿酸沉积血管内膜 → 内皮损伤 → **动脉粥样硬化加速**
- β细胞长期超负荷 → 功能衰竭 → **进展为2型糖尿病**

### 4. 发出风险预警
- 提示当前处于**糖尿病前期**或**心血管高危状态**
- 建议干预重点：减重、改善胰岛素敏感性、控制血压血脂
