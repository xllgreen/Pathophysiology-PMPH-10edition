# 氨对脑神经递质的双重干扰机制

## 何时使用
当满足以下条件时应用本技能：
- 血氨水平高于正常值
- 患者出现意识模糊、扑翼样震颤、躁动或昏迷等神经精神症状
- 临床怀疑为肝性脑病（排除非氨中毒性脑病，如无高氨的代谢性脑病）

## 执行步骤

### 1. 分析谷氨酸能系统的变化
- **早期阶段**：评估是否因氨选择性抑制α-酮戊二酸脱氢酶（α-KGDH）但未显著抑制丙酮酸脱氢酶（PD），导致α-酮戊二酸蓄积，进而通过转氨基作用生成谷氨酸增多，引起兴奋性增强（如躁动）。
- **晚期阶段**：判断氨是否极度升高，同时抑制PD和α-KGDH，导致三羧酸循环受抑、谷氨酸合成减少；同时检查星形胶质细胞中谷氨酰胺合成是否增强（氨 + 谷氨酸 → 谷氨酰胺），造成谷氨酸耗竭。
- 注意谷氨酰胺的双重作用：具有类抑制性递质效应，并可诱导星形胶质细胞肿胀及自由基生成。
- 若观察到全脑谷氨酸水平下降但患者仍表现兴奋，需考虑突触间隙谷氨酸可能因钙依赖性过度释放或EAAT2表达下调而局部升高。

### 2. 评估抑制性递质系统的增强
- 检查GABA和甘氨酸活性是否增强。
- 判断氨是否上调线粒体外膜PTBR，促进孕烯醇酮合成，进而增加神经类固醇（如THP、THDOC），作为GABA-A受体变构激动剂增强抑制效应。
- 评估星形胶质细胞对GABA的摄取是否减少、释放是否增加，导致突触间隙GABA浓度上升。

### 3. 检查其他递质变化
- 确认乙酰胆碱是否因PD受抑导致乙酰辅酶A减少而合成下降，从而减弱兴奋性。
- 注意多巴胺和去甲肾上腺素水平是否随疾病进展发生改变。

### 4. 综合判断
整合上述变化，得出结论：**兴奋性递质（谷氨酸、乙酰胆碱）减少 + 抑制性递质（谷氨酰胺、GABA）增加 → 中枢神经功能紊乱**，解释患者从兴奋状态向抑制乃至昏迷的演变过程。

> **注意**：本机制仅适用于高氨相关的肝性脑病，不适用于无高氨的其他类型脑病。
