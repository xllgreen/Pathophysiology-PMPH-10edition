# 受体酪氨酸激酶（RTK）信号转导机制

## 何时使用
- 分析EGFR、PDGFR、VEGFR等RTK如何响应生长因子刺激
- 解释RTK二聚化、自身磷酸化及SH2结构域蛋白招募过程
- 追踪RAS-MAPK、PI3K-Akt或PLCγ-PKC通路的激活逻辑
- 区分RTK通路与GPCR等其他膜受体信号机制

## 执行步骤
1. **确认配体结合与受体二聚化**：验证生长因子（如EGF、PDGF、VEGF）与RTK胞外区结合，诱导同源或异源二聚化。
2. **激活PTK活性**：二聚化激活RTK胞内蛋白酪氨酸激酶（PTK）活性，催化自身酪氨酸残基磷酸化。
3. **招募下游信号蛋白**：含SH2结构域的蛋白（如GRB2、PLCγ、PI3K）识别磷酸化酪氨酸位点并结合。
4. **启动主要下游通路**：
   - **RAS-MAPK通路**：GRB2-SOS激活RAS → RAF → MEK → ERK → 调控基因表达与细胞增殖
   - **PI3K-PKB通路**：PI3K生成PIP₃ → 激活PKB（Akt）→ 促进细胞存活与代谢
   - **PLC-PKC通路**：PLCγ水解PIP₂ → 生成IP₃/DAG → 激活PKC
5. **输出结果**：明确对细胞增殖、分化、免疫应答或肿瘤发生的影响。

> 注意：仅适用于具有PTK活性且依赖二聚化的RTK；不适用于无PTK活性或非二聚化受体。
