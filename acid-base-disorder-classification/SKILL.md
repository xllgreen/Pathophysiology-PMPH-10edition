# 酸碱平衡紊乱分类诊断

## 适用场景
- 已获取患者动脉血气结果（含pH、PaCO₂、HCO₃⁻）
- 临床怀疑存在酸碱失衡
- 需区分原发性紊乱与代偿反应

## 执行步骤
1. **确认基本参数**：确保pH、PaCO₂、HCO₃⁻数据完整且可靠。
2. **识别原发性紊乱**：
   - 判断是否存在代谢性酸中毒、代谢性碱中毒、呼吸性酸中毒或呼吸性碱中毒
   - 排除代偿性变化（代偿是继发反应，不计入紊乱类型）
3. **计数原发性紊乱数量**：
   - 仅1种 → 归类为“单纯型酸碱平衡紊乱”
   - 2种同时存在 → “双重性酸碱失衡”
   - 3种（罕见）→ “三重性酸碱失衡”
4. **结合临床背景验证**：例如COPD患者合并呕吐，可能表现为呼吸性酸中毒+代谢性碱中毒（双重性）。
5. **输出分类结果**：明确列出紊乱类型及数量，用于指导后续病因排查与治疗。

## 注意事项
- 不适用于未进行血气分析的初步筛查
- 必须区分代偿与原发紊乱，避免误判
- 罕见三重失衡需高度警惕混合病因（如肾衰+腹泻+机械通气不当）
