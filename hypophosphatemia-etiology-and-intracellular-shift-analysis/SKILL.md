# 低磷血症成因与细胞内转移机制分析

## 触发条件
当血清无机磷浓度 < 0.8 mmol/L，并已排除实验室误差及暂时性波动时，启动本分析流程。

## 执行步骤

1. **确认诊断标准**：血清无机磷 < 0.8 mmol/L（正常成人范围：0.8–1.6 mmol/L）。

2. **分类评估三大机制**：
   - **A. 小肠磷吸收减低**：排查长期饥饿、呕吐、腹泻、1,25-(OH)₂D₃不足、吸收不良综合征，或近期使用磷酸结合剂（如氢氧化铝凝胶、碳酸铝、氢氧化镁）。
   - **B. 尿磷排泄增加**：评估是否存在急性乙醇中毒、原发/继发性甲旁亢、肾小管性酸中毒、Fanconi综合征、维生素D抵抗性佝偻病、代谢性酸中毒、糖尿病，或使用糖皮质激素、利尿剂。
   - **C. 磷向细胞内转移**（关键机制）：检查是否因胰岛素、雄激素、静脉输注葡萄糖/果糖促进合成代谢；是否处于再喂养综合征；或存在呼吸性碱中毒（激活磷酸果糖激酶，导致无机磷消耗）。

3. **评估临床影响**：结合ATP合成不足与红细胞2,3-DPG减少，判断症状严重程度（从无症状到肌无力、感觉异常、骨痛、精神障碍甚至昏迷）。

4. **输出结果**：明确低磷血症类型及潜在病因，为补磷治疗和原发病管理提供依据。

> 注意：所有分析必须基于已确认的血清无机磷测定结果，并排除非病理性波动。
