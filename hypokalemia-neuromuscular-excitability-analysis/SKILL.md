# 低钾血症神经肌肉兴奋性分析

## 何时使用
- 血清钾浓度 < 3.5 mmol/L
- 出现下肢软弱、弛缓性瘫痪、腹胀、便秘或麻痹性肠梗阻
- 需判断是否为急性低钾血症以评估紧急程度

## 执行步骤
1. **判断病程性质**：
   - 急性：血钾迅速下降
   - 慢性：血钾缓慢降低
2. **分析膜电位变化**：
   - **急性低钾**：
     - [K⁺]ₑ↓ → [K⁺]ᵢ/[K⁺]ₑ↑ → K⁺外流↑ → 静息电位超极化
     - Em 与 Et 间距加大 → 兴奋性↓ → “超极化阻滞”
     - 可致严重肌无力、肠麻痹、嗜睡甚至昏迷
   - **慢性低钾**：
     - 细胞内钾外移代偿 → 静息电位基本正常 → 症状轻微
3. **预警严重后果**：急性低钾血症可导致呼吸肌麻痹或肠穿孔，需紧急补钾。

> 注意：该机制主要影响可兴奋组织，需排除其他电解质紊乱干扰。
