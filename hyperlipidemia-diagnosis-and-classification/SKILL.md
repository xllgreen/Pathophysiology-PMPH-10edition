# 高脂血症诊断与分类

## 适用对象
- 成人患者
- 临床血脂检测结果

## 前提条件
- 患者处于空腹状态
- 已获得血浆总胆固醇（TC）和甘油三酯（TG）检测数据

## 执行步骤
1. 获取空腹血样，测定 TC、TG 和 HDL-C 浓度。
2. 应用中国诊断阈值判断是否存在高脂血症：
   - **高胆固醇血症**：TC ≥ 6.2 mmol/L（240 mg/dl）
   - **高甘油三酯血症**：TG ≥ 2.3 mmol/L（200 mg/dl）
   - **混合型高脂血症**：TC 和 TG 均升高
3. 同时评估 HDL-C：<1.0 mmol/L（40 mg/dL）视为低HDL-C血症，需并列处理。
4. 进行多维分类：
   - **病因分类**：原发性（遗传） vs 继发性（如糖尿病、肾病、药物）
   - **临床分类**：高胆固醇血症、高甘油三酯血症、混合型
   - **表型分类**（参考WHO 1970年六型系统）：I、IIa、IIb、III、IV、V型

## 注意事项
- 不适用于儿童或非空腹状态下的检测结果。
- 高脂血症即高脂蛋白血症，因血脂以脂蛋白形式运输。

## 输出结果
返回高脂血症类型（如“高胆固醇血症”）、是否合并低HDL-C血症，并提示可能的病因方向，用于指导后续筛查与治疗策略。
