# 阿尔茨海默病蛋白质异常聚集致病机制分析

## 适用对象
- 阿尔茨海默病患者
- 携带APP/PS1/PS2突变个体
- tau蛋白过度磷酸化模型

## 触发条件
当出现以下任一情况时启用本技能：
- Aβ或tau蛋白发生异常聚集
- 需要解析AD相关基因突变（如APP、PSEN1、PSEN2）的致病路径
- 评估蛋白质降解系统功能下降对神经元的影响

## 执行流程

### 步骤1：分析Aβ异常聚集机制
1. 确认Aβ生成与清除是否失衡。
2. 检查是否存在以下基因突变：
   - APP突变 → 改变酶切位点，增加Aβ产量；
   - PS1/PS2突变 → 增强γ-分泌酶活性，提高Aβ42比例。
3. 评估辅助因子异常（如APOE ε4、α-2巨球蛋白）对Aβ清除的影响。
4. 判断Aβ寡聚体是否引发以下毒性效应：
   - 破坏Ca²⁺稳态；
   - 诱导氧化应激；
   - 促进tau蛋白过度磷酸化。

### 步骤2：分析tau蛋白异常修饰与聚集
1. 检测tau蛋白是否发生过度磷酸化、糖基化或泛素化。
2. 评估磷酸化tau与总tau的比值（p_tau_ratio）。
3. 判断tau是否从微管解离并形成神经原纤维缠结（NFT）。
4. 分析NFT对以下细胞结构的影响：
   - 微管稳定性下降；
   - 轴浆转运障碍；
   - 突触功能受损。

### 步骤3：整合双重机制结论
综合Aβ斑块负荷与tau缠结程度，判断神经元损伤阶段，并识别潜在治疗靶点。

## 注意事项
- 本技能不适用于非聚集性蛋白病（如朊病毒病）。
- 必须确认存在基因突变或翻译后修饰异常作为前提。
- 蛋白质降解系统功能下降是必要条件之一。
