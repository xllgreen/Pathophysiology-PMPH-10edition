# 低镁血症神经肌肉与心血管影响评估

## 何时使用
- 患者血清镁浓度 < 0.75 mmol/L
- 出现肌肉震颤、手足搐搦、反射亢进、焦虑或室性心律失常
- 需与单纯低钙血症鉴别

## 执行步骤
1. **确认低镁状态**：血清镁 < 0.75 mmol/L。
2. **分析神经肌肉机制**：
   - Mg²⁺减少 → Ca²⁺内流增多 → 乙酰胆碱释放增加
   - 终板膜乙酰胆碱受体抑制减弱 → 兴奋传递增强
   - 神经与骨骼肌兴奋性抑制作用减弱
3. **评估心血管影响**：
   - 心肌静息电位绝对值变小 → 兴奋性增高
   - 自律性增高（钠内流阻断减弱）
   - Na⁺-K⁺泵活性下降 → 细胞内缺钾 → 室性心律失常风险升高
4. **识别关联表现**：高血压（外周阻力↑）、冠脉痉挛、胃肠道平滑肌兴奋（呕吐/腹泻）。
5. **启动干预**：补镁，同时监测心电图、血压及伴随的低钾/低钙。

> 警示：严重低镁可致癫痫、昏迷或室颤猝死，需紧急处理。
