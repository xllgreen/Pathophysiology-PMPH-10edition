# 氨基酸失衡促进肝性脑病的机制分析

## 何时使用
- 患者为肝硬化或门体分流术后个体；
- 血氨升高且存在胰岛素/胰高血糖素比例失调；
- 实验室检测显示血浆BCAA/AAA比值 < 1.2。

## 执行步骤

### 1. 确认氨基酸失衡表现
- 验证血浆中芳香族氨基酸（AAA：苯丙氨酸、酪氨酸）升高；
- 验证支链氨基酸（BCAA：亮氨酸、异亮氨酸、缬氨酸）降低；
- 计算BCAA/AAA摩尔比，确认其低于正常范围（正常为3~3.5，肝性脑病时降至0.6~1.2）。

### 2. 分析失衡成因
- 评估肝脏灭活胰岛素和胰高血糖素能力下降导致的激素比例变化；
- 判断肝脏对AAA清除减少及糖异生障碍的影响；
- 考察高胰岛素水平促进肌肉摄取BCAA的作用；
- 分析高血氨如何通过转氨基反应消耗BCAA生成谷氨酰胺。

### 3. 推演脑内后果
- 确认AAA与BCAA竞争通过血脑屏障的电中性氨基酸转运体；
- 评估AAA入脑增加对神经递质合成的影响：
  - 高苯丙氨酸抑制酪氨酸羟化酶 → 多巴胺、去甲肾上腺素合成减少；
  - 苯丙氨酸和酪氨酸分别转化为苯乙醇胺和羟苯乙醇胺（假性神经递质）。

### 4. 临床决策支持
- 若BCAA/AAA比值显著降低，可考虑支链氨基酸补充以缓解症状；
- 注意：该干预不能提高生存率，仅作为对症支持；
- 将此机制视为假性神经递质学说的补充，而非独立病因。

> **注意**：该失衡可能是氨中毒的结果，而非原发病因。需结合血氨水平及其他肝性脑病指标综合判断。
