# 脂蛋白分类、组成与代谢功能

## 何时使用
- 解读血脂检测结果（如总胆固醇、甘油三酯、HDL-C、LDL-C）
- 评估动脉粥样硬化或心血管疾病风险
- 分析脂代谢紊乱（如高脂血症、脂肪肝）的病理机制
- 判断载脂蛋白表达异常对脂质运输的影响

> **注意**：本技能不适用于脂蛋白合成障碍遗传病（如无β脂蛋白血症）或严重肝衰竭患者。

## 执行步骤

1. **确认触发条件**：体内存在胆固醇或甘油三酯运输需求，且载脂蛋白正常表达。

2. **识别脂蛋白基本结构**：
   - 球形颗粒，含疏水核心（胆固醇酯、甘油三酯）和亲水外壳（磷脂、游离胆固醇、载脂蛋白）。

3. **按密度分类并匹配功能**：
   - **乳糜微粒（CM）**：富含甘油三酯（TG），载脂蛋白为ApoB48、ApoAI、AII；从小肠转运饮食来源TG至外周组织。
   - **极低密度脂蛋白（VLDL）**：富含内源性TG，载脂蛋白为ApoB100、E、Cs；由肝脏合成，向全身输送能量。
   - **中间密度脂蛋白（IDL）**：VLDL残粒，含TG与胆固醇酯（CE），载脂蛋白为ApoB100、E；部分被肝脏清除。
   - **低密度脂蛋白（LDL）**：富含CE和磷脂（PL），仅含ApoB100；是胆固醇主要载体，经LDL受体被外周组织摄取。
   - **高密度脂蛋白（HDL）**：富含PL和CE，载脂蛋白为ApoAI、AII、Cs；介导胆固醇逆转运（RCT），具有抗动脉粥样硬化作用。
   - **脂蛋白(a) [Lp(a)]**：Apo(a)通过二硫键连接LDL；水平升高与动脉粥样硬化正相关。

4. **分析载脂蛋白关键作用**：
   - 形成水溶性复合物
   - 作为细胞受体配体（如ApoB100识别LDL受体）
   - 调节脂代谢酶活性（如ApoCII激活脂蛋白脂肪酶LPL，ApoAI激活卵磷脂胆固醇酰基转移酶LCAT）

5. **输出临床判断**：
   - 根据脂蛋白谱异常模式（如LDL升高、HDL降低、Lp(a)升高）判断脂代谢状态
   - 指导动脉粥样硬化风险分层与干预策略
