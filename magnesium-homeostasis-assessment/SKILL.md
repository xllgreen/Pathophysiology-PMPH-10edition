# 镁稳态评估

## 适用条件
- 适用于具有正常肾功能和正常饮食摄入的成年个体
- 不适用于严重肾衰竭患者或长期接受未补充镁的静脉营养者

## 执行步骤
1. 确认个体满足前提条件（正常肾功能、正常饮食）。
2. 获取以下关键参数：
   - 体内镁总量（正常范围：21~28 克）
   - 血清镁浓度（正常范围：0.75~1.25 mmol/L）
   - 每日镁摄入量（成人通常为 10~20 mmol）
3. 分析镁的体内分布：
   - 约 60% 储存于骨骼（以 Mg₃(PO₄)₂ 和 MgCO₃ 形式吸附于羟基磷灰石表面）
   - 约 38%~39% 存在于骨骼肌及其他组织细胞内
   - 仅 1%~2% 分布于细胞外液（含血清）
4. 评估肠道吸收情况：每日摄入镁中约 1/3（3.3~6.7 mmol）在小肠吸收，其余经粪便排出。
5. 判断细胞内镁状态：主要与磷酸根、柠檬酸根结合，尤其以 Mg·ATP 复合物形式参与能量代谢。
6. 综合上述信息，判断个体是否处于镁稳态，并作为诊断电解质紊乱的基准。

> 提示：骨中镁不易被快速动员，但可在骨钙动员过程中发生部分置换，置换程度取决于骨代谢活跃度。
