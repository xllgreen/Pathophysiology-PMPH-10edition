# 脑脊液pH对脑电活动影响的阈值评估

## 何时使用
- 患者出现意识障碍、抽搐或其他神经系统症状，并伴有已知或疑似呼吸衰竭或酸中毒；
- 脑脊液pH值可获取（如通过腰椎穿刺）；
- 需要判断中枢神经系统抑制程度以指导临床干预时机。

## 执行步骤
1. 获取脑脊液样本并准确测量其pH值。
2. 根据以下阈值判断脑电活动状态：
   - 若 pH ≥ 7.25：脑电活动正常或仅轻度受影响；
   - 若 6.8 ≤ pH < 7.25：脑电波变慢，提示中枢抑制开始；
   - 若 pH < 6.8：脑电活动完全停止，提示严重脑功能障碍甚至不可逆损伤风险。
3. 注意区分脑脊液pH与动脉血pH——脑脊液缓冲能力弱、调节慢，在呼吸衰竭时其pH变化比血液更显著，因此不能直接用血液pH替代。
4. 结合临床表现和其他检查结果，综合评估是否需紧急纠正酸中毒或支持脑功能。

> **警告**：此技能仅适用于中枢神经系统相关评估，不适用于外周组织或非神经系统的酸碱失衡分析。
