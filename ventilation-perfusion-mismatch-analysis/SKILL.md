# 肺泡通气血流比例失调分析

## 何时使用
在以下情境中启用本技能：
- 患者有肺部疾病（如肺炎、肺栓塞、肺纤维化）
- 动脉血气显示PaO₂显著降低，而PaCO₂正常、降低或轻度升高
- 怀疑气体交换障碍源于局部V̇A/Q̇比例异常，而非整体通气或循环衰竭

## 执行步骤
1. **确认基础参数**：正常静息状态下，V̇A ≈ 4 L/min，Q̇ ≈ 5 L/min，V̇A/Q̇ ≈ 0.8。
2. **识别失调类型**：
   - **V̇A/Q̇ ↑（>0.8）**：通气相对过剩（如肺栓塞导致血流减少）
   - **V̇A/Q̇ ↓（<0.8）**：通气不足或血流相对过多（如气道阻塞、肺实变）
3. **评估生理影响**：即使总V̇A和Q̇正常，局部V̇A/Q̇显著偏离0.8仍可导致严重低氧血症。
4. **关联临床表现**：V̇A/Q̇失调是肺部疾病引起Ⅰ型或Ⅱ型呼吸衰竭的最常见机制。
5. **排除理想模型**：本分析不适用于无肺实质病变或通气/血流完全均匀的假设情况。

> 提示：年龄增长和体位（如直立位）可生理性扩大V̇A/Q̇区域差异，需结合病理背景综合判断。
