# 呼吸性碱中毒对神经系统影响的评估

## 何时使用
- 患者已确诊呼吸性碱中毒（动脉血气显示PaCO₂↓、pH↑）；
- PaCO₂快速或显著下降（例如降低20mmHg或更多）；
- 出现神经系统症状（如眩晕、口周或四肢麻木、抽搐、意识模糊）。

## 执行步骤
1. **识别典型神经系统症状**：
   - 注意呼吸性碱中毒比代谢性碱中毒更易引发眩晕、四肢及口周感觉异常、意识障碍甚至抽搐。

2. **评估抽搐是否与低钙相关**：
   - 碱中毒使白蛋白结合钙增加，导致血浆游离Ca²⁺浓度下降；
   - 若出现手足搐搦，应立即检测离子钙（Ca²⁺_free）。

3. **判断脑功能障碍机制**：
   - 碱中毒直接损害脑细胞功能；
   - 低碳酸血症引起脑血管收缩，显著减少脑血流量；
   - 参考数据：PaCO₂每下降20mmHg，脑血流量减少35%~40%。

4. **临床警示**：
   - 即使pH仅轻度升高，若PaCO₂骤降，仍可导致严重神经症状；
   - 需警惕并优先处理潜在危象。

> **输出提示**：警惕神经系统症状，尤其当PaCO₂快速下降时；若出现手足搐搦，应考虑低钙并检测离子钙。
