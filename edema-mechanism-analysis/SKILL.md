# 水肿机制分析

## 何时使用
- 患者出现局部或全身性水肿
- 需区分水肿类型以制定针对性治疗（如利尿、补蛋白、抗炎或引流）
- 排除单纯细胞内水肿（如水中毒）

## 执行步骤
1. **确认触发条件**：存在组织液生成与回流动态失衡，且淋巴代偿能力超限。
2. **计算有效滤过压**（参考值）：
   - 动脉端 ≈ +10 mmHg（促进滤出）
   - 静脉端 ≈ -8 mmHg（促进回流）
3. **逐一排查四大机制**：
   - **毛细血管流体静压增高**：见于心衰（全身水肿）、静脉压迫（局部水肿）、动脉充血（炎症）
   - **血浆胶体渗透压降低**：白蛋白<25 mmHg，见于肝硬化、肾病综合征、营养不良
   - **微血管通透性增加**：感染、烧伤等导致血浆蛋白漏出，水肿液蛋白含量高（30~60 g/L）
   - **淋巴回流受阻**：肿瘤、手术、丝虫病致淋巴堵塞，水肿液蛋白含量高（40~50 g/L）
4. **结合临床判断主导机制**，选择对应治疗策略。

> 注意：必须结合体内外水钠平衡综合评估，不可孤立分析。
