# MODS双相发病类型区分

## 适用条件
- 患者已确诊MODS（≥2个器官系统功能障碍）
- 存在明确的初始打击事件（如严重创伤、大出血、休克等）
- **不适用于**慢性器官衰竭基础上的急性加重（如肝肾综合征）

## 执行步骤

1. **判断是否存在“缓解期”**：
   - 若初始打击后病情一度稳定（持续数天），随后再次恶化 → 考虑**双相迟发型**
   - 若无缓解期，器官功能障碍在数小时至1–2天内迅速出现 → 考虑**单相速发型**

2. **分析第二个器官衰竭的诱因**：
   - 是否由新发感染、脓毒症或其他继发性炎症事件（“二次打击”）触发？
     - 是 → 支持**双相迟发型**
     - 否（直接由原始损伤导致）→ 支持**单相速发型**

3. **确认发病模式**：
   - **单相速发型（原发型/一次打击型）**：
     - 由原始损伤因子直接引起
     - 病情发展快，仅有一个病变高峰
     - 常见于严重创伤、大面积烧伤、休克复苏早期
   - **双相迟发型（继发型/二次打击型）**：
     - 初始损伤后经历相对稳定期
     - 因继发事件（如全身性感染）触发第二次打击
     - 出现第二个器官功能障碍高峰，死亡风险更高

4. **输出结果**：
   - 明确MODS类型（单相速发型 或 双相迟发型）
   - 据此调整临床干预重点：
     - 单相型：聚焦早期控制原发损伤
     - 双相型：加强感染防控、免疫监测与二次打击预防

## 触发场景
当用户描述危重患者出现多器官衰竭，并提及初始创伤/感染事件及后续病情变化时，调用本技能进行分型判断。
