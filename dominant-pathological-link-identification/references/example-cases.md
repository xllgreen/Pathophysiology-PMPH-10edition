# 典型案例参考

## 案例：甲状腺功能亢进引发的心血管并发症

**病理链**：
甲状腺功能亢进（甲亢） → 过量甲状腺激素 → 
- 心肌代谢异常
- 血流动力学改变
- 交感-肾上腺髓质系统激活
- 肾素-血管紧张素-醛固酮系统（RAAS）紊乱 →
心律失常 → 心绞痛 / 心肌梗死 / 心力衰竭

**主导环节**：甲状腺功能亢进

**错误做法**：仅使用抗心律失常药物控制房颤
**正确策略**：优先控制甲亢（如抗甲状腺药物、放射性碘或手术），心律失常常随之缓解

## 其他适用场景
- 肝硬化导致门脉高压 → 腹水、食管静脉曲张 → 上消化道出血（主导环节：肝硬化）
- 慢性肾病 → 继发性甲旁亢 → 骨病与血管钙化（主导环节：肾功能衰竭）

## 不适用场景示例
- 单纯细菌性肺炎（单一病因，无级联）
- 急性踝关节扭伤（局部损伤，无系统性连锁反应）
