# 识别疾病主导环节并制定针对性治疗策略

## 何时使用
- 疾病涉及多个器官系统或病理过程
- 存在明显的因果级联反应（如A→B→C）
- 患者表现多样但可能源于单一核心病因
- 常规对症治疗效果不佳或病情反复

## 执行步骤
1. **绘制病理因果链图**：列出疾病过程中所有已知的病理环节，并标注其时间顺序与逻辑依赖关系。
2. **识别主导环节**：判断哪个环节满足以下全部条件：
   - 是其他环节发生的必要前提
   - 其存在直接驱动后续病理变化
   - 若被纠正或去除，可有效阻断整个病理进程
3. **确认治疗靶点**：将干预重点放在主导环节，而非继发表现（如心律失常、水肿等）。
4. **制定治疗方案**：围绕主导环节选择病因治疗、功能调控或源头阻断措施。
5. **评估可逆性**：若主导环节已不可逆（如终末期器官纤维化），则调整策略，转向控制关键继发环节以延缓进展。

> **注意**：本技能不适用于病因单一、无级联反应的疾病（如单纯外伤或孤立感染）。

## 输出目标
制定以主导环节为核心的治疗方案，提高疗效、减少无效干预，并防止病情进一步恶化。
