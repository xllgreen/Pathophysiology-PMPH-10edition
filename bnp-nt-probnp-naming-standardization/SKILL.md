# B型钠尿肽及其前体的临床命名规范

- 编写或审核涉及心力衰竭诊断、心血管疾病生物标志物或临床检验报告的正式医学文本；
- 首次引入“B型钠尿肽”或“N末端B型钠尿肽原”术语时；

1. **识别目标术语**：确认文本中是否涉及以下两类生物标志物之一：

   - 中文术语 + 英文全称（括号内）+ 标准缩写（括号内）
3. **后续使用**：首次完整标注后，可在同一文档中使用对应标准缩写。
4. **适用范围**：仅用于正式医学语境（如临床指南、科研论文、检验报告），不适用于非专业文献或非标准化缩写场景。

> 注意：务必区分BNP与NT-proBNP，二者不可混用或简写为相同形式。
