# 病理生理学定义与核心任务

## 何时使用
- 研究对象为疾病状态下机体的功能、代谢和结构变化
- 需要解释“为什么”出现某种病理表现（如缺氧为何导致乳酸堆积）
- 制定诊断、治疗或预防策略需理论支撑
- 区分病理生理学与病理学、内科学等邻近学科

## 执行步骤
1. **明确学科定义**：病理生理学是研究疾病发生原因、发展机制及转归过程中功能和代谢变化规律的学科。
2. **聚焦三大变化层面**：
   - 功能改变（如心输出量下降）
   - 代谢紊乱（如酸碱失衡）
   - 结构损伤（如细胞坏死）
3. **强调机制导向**：不仅描述现象，更要揭示内在因果链条（例如无氧酵解增强→乳酸堆积）。
4. **整合基础医学知识**：衔接解剖学、生理学、生物化学、免疫学，构建疾病整体观。
5. **服务临床实践**：为症状分析、实验室指标解读、治疗靶点发现提供依据。
6. **明确学科边界**：
   - ≠ 病理学（后者侧重形态学）
   - ≠ 内科学（后者侧重具体疾病诊疗）
