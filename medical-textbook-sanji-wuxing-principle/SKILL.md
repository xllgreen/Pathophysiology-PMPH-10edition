# 医学教材“三基五性”编写原则

## 何时使用
- 编写或审阅五年制本科临床医学专业国家规划教材
- 评估医学教材是否符合官方标准

## 执行步骤
1. **落实“三基”要求**：
   - 基础理论：如稳态失衡机制
   - 基本知识：如应激、细胞死亡等共性病理过程
   - 基本技能：如通过病例分析训练机制推理能力
2. **贯彻“五性”标准**：
   - 思想性：融入课程思政，弘扬医者精神
   - 科学性：内容经院士专家审定，确保准确
   - 先进性：纳入新进展（如“细胞死亡”替代“凋亡”，新增“衰老与疾病”）
   - 启发性：通过思维导图、病例分析激发思考
   - 适用性：控制字数（约53万字），采用双色/全彩印刷，配套数字资源
3. **执行多轮修订流程**：初稿讨论 → 交叉审稿 → 主编再审 → 定稿会
4. **配套数字资源**：扫码获取PPT、微课、动画等，拓展内容而不增加纸质篇幅
