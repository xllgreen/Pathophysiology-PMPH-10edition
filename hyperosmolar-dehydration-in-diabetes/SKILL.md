# 糖尿病高渗性脱水与渗透性利尿机制

## 何时使用
- 患者血糖 >10 mmol/L 并出现多尿、极度口渴、意识模糊
- 实验室检查显示高血浆渗透压但无明显酮症
- 需解释为何高血糖导致脱水而非水肿

## 执行步骤

### 1. 确认触发条件
- 血糖浓度 > 肾糖阈（通常≈10 mmol/L）
- 存在胰岛素功能缺陷（如2型糖尿病失代偿）

### 2. 分析病理过程
- 高血糖 → 细胞外液渗透压↑ → 水从细胞内移至细胞外 → **细胞内脱水**
- 脑细胞脱水 → 可致**高渗性非酮症糖尿病昏迷**
- 肾小球滤过葡萄糖 > 肾小管重吸收能力 → 小管液葡萄糖↑ → 渗透压↑ → **抑制水重吸收** → **渗透性利尿**

### 3. 识别关键特征
- **多尿、烦渴、体重下降**
- **糖尿阳性**
- **无显著酮体生成**（区别于DKA）
- 血钠可能假性正常或升高（因水丢失）

### 4. 发出临床警示
- 提示高渗状态风险，需紧急：
  - 监测血浆渗透压（>320 mOsm/kg为危象）
  - 缓慢降糖 + 补液纠正脱水
  - 避免快速纠正以防脑水肿
