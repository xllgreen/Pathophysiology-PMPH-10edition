# Pathophysiology Core Skills — PMPH 10th Edition

This skill collection comprises **145 core competencies** in pathophysiology, systematically organized to bridge basic medical sciences and clinical practice.

## Structure

Each skill is a directory containing a `SKILL.md` file with YAML frontmatter and structured markdown content. Skills are organized by topic category and can be invoked by name.

## Usage

Refer to individual skills by their directory name:

- `edema-fluid-classification` — classify transudate vs exudate
- `acid-base-disorder-classification` — classify acid-base disorders via blood gas
- `dic-pathogenesis-assessment` — assess DIC pathogenesis
- `hypoxic-pulmonary-vasoconstriction-mechanism` — analyze HPV mechanism
- ... and 140+ more skills

See [index.md](index.md) for the complete skill matrix.
