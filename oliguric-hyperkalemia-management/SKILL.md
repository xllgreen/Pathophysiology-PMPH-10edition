# 少尿期高钾血症处理

## 何时使用
当满足以下全部条件时启动本技能：
- 患者处于少尿型急性肾损伤（ARF）的少尿期
- 血钾浓度 > 5.5 mmol/L 或 出现高钾相关心电图改变（如T波高尖、QRS增宽）

**注意**：不适用于多尿期、恢复期或非少尿型ARF患者。

## 执行步骤
按紧急程度依次实施以下措施：

1. **立即限制钾摄入**：
   - 禁食高钾食物（香蕉、橙汁、土豆等）
   - 停用保钾药物（如螺内酯、ACEI/ARB等）

2. **促进钾向细胞内转移**：
   - 静脉给予10单位胰岛素 + 50 ml 50%葡萄糖溶液，缓慢滴注

3. **对抗心脏毒性（如有心电图异常）**：
   - 缓慢静脉注射10%葡萄糖酸钙10 ml（2~5分钟），稳定心肌细胞膜

4. **促进肠道排钾**：
   - 口服或灌肠钠型阳离子交换树脂（如聚苯乙烯磺酸钠）

5. **紧急透析指征**：
   - 血钾 > 6.5 mmol/L
   - 或出现严重心律失常、QRS波显著增宽
   - 立即安排血液透析

6. **监测要求**：
   - 每2~4小时复查血钾和心电图，直至稳定

## 输出结果
根据血钾水平和心电图表现，输出具体干预措施组合，并明确是否需紧急透析。
