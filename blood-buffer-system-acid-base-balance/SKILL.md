# 血液缓冲系统在酸碱平衡中的作用机制

## 何时使用
- 出现急性酸中毒或碱中毒（如乳酸酸中毒、酮症酸中毒、过度通气等）
- 需要解释动脉血气分析中pH、[HCO₃⁻]和PaCO₂三者关系
- 区分挥发酸（如CO₂）与固定酸（如乳酸、酮体）的缓冲路径
- 理解为何仅靠缓冲无法纠正慢性酸碱紊乱

## 核心操作流程
1. **识别主要缓冲系统**：确认五大缓冲对及其贡献比例——碳酸氢盐（53%）、血红蛋白系统（35%）、血浆蛋白（7%）、磷酸盐（5%）。
2. **判断酸类型**：
   - 若为**固定酸**（非挥发性），主要由碳酸氢盐系统缓冲；
   - 若为**挥发酸**（CO₂↑），主要由血红蛋白系统缓冲。
3. **应用Henderson-Hasselbalch方程**：计算理论pH = 6.1 + lg([HCO₃⁻]/(0.03 × PaCO₂))，验证实测pH是否符合预期。
4. **评估缓冲方向**：
   - H⁺增多 → 缓冲反应左移 → [HCO₃⁻]下降；
   - H⁺减少 → 反应右移 → [HCO₃⁻]相对升高。
5. **强调局限性**：指出缓冲能力有限，必须结合呼吸（调节PaCO₂）和肾脏（调节[HCO₃⁻]）进行长期代偿。

> 注意：磷酸盐和血浆蛋白缓冲作用较小，通常在其他系统耗竭后才显著参与。
