# 代谢性酸中毒心血管影响评估

## 何时使用
- 已诊断代谢性酸中毒
- 出现高血钾、室性心律失常、低血压或休克难以纠正
- 需判断酸中毒是否加重循环障碍

## 执行步骤
1. **确认酸中毒状态**：存在H⁺浓度升高。
2. **分析三大机制**：
   - **高钾血症形成**：
     - H⁺入胞 → K⁺出胞 → 血钾↑
     - 肾排K⁺减少 → 进一步升高血钾 → 诱发传导阻滞、室颤
   - **心肌收缩力下降**：
     - H⁺竞争抑制Ca²⁺与肌钙蛋白结合
     - 干扰Ca²⁺内流及肌质网释放
   - **血管低反应性**：
     - 外周血管（尤其毛细血管前括约肌）对儿茶酚胺敏感性↓
     - 血管扩张 → 回心血量↓ → 血压↓
3. **启动紧急干预**：若出现上述表现，必须优先纠正酸中毒，否则休克难以逆转。

> 警示：重度代谢性酸中毒可直接导致心搏停止，需联合纠酸、降钾、支持循环。
