## About This Project

This project systematically integrates **145 core competencies** in pathophysiology, covering acid-base balance disorders, shock and multiple organ dysfunction, water and electrolyte metabolism abnormalities, inflammation and immune response, cell signal transduction, tumorigenesis mechanisms, aging-related pathological processes, respiratory and circulatory dysfunction, neuropsychiatric disorders, renal pathophysiology, endocrine and metabolic diseases, and other key topics.

**Target Audience**: Medical students, pathophysiology educators, clinicians, basic medical researchers

**Reference Textbook**: Pathophysiology, 10th Edition, People's Medical Publishing House (PMPH)

**⚠️ Risk ⚠️**: This skill set covers disease mechanism analysis, diagnostic reasoning, and treatment principles, which could be misused as independent clinical decision-making basis.

Mitigation: Use outputs only as educational or clinician-reviewed reference material, and verify recommendations against current official guidelines, local protocols, and qualified expert opinion.

## Quick Start

### Installation

CLI:
```bash
openclaw skills install pathophysiology-pmph-10edition
```

### Usage

Each skill contains four sections:
1. **When to Use** — triggers for the skill
2. **Execution Steps** — standardized operating procedure
3. **Cautions** — contraindications and warnings
4. **References** — detailed supplementary materials

### Query Examples

#### 1. Concept Understanding
```
Please explain the physiological significance of hypoxic-pulmonary-vasoconstriction-mechanism and its changes in ARDS.
```

#### 2. Mechanism Analysis
```
How to apply the dic-diagnosis-and-classification criteria for clinical judgment?
```

#### 3. Comprehensive Reasoning
```
Analyze the risk of sudden cardiac death in stress cardiomyopathy using stress-induced-cardiac-sudden-death-mechanism.
```
