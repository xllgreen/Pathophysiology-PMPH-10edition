# 夜间阵发性呼吸困难识别与评估

## 何时使用
当遇到以下情况时立即启用本技能：
- 患者在夜间入睡1~2小时后突然因气闷、气急惊醒，被迫坐起
- 伴有咳嗽或咳出泡沫样痰（严重时为粉红色泡沫痰）
- 已知存在左心衰竭或全心衰竭病史，且已排除肺部原发疾病

## 执行步骤
1. **确认典型表现**：判断是否符合夜间阵发性呼吸困难的定义——夜间平卧后突发呼吸困难，坐起可缓解，重者进展为急性肺水肿。
2. **分析发生机制**：
   - 平卧位导致下半身静脉回流增加及水肿液吸收入血，加重肺淤血
   - 夜间迷走神经兴奋性增高引起小支气管收缩，增加气道阻力
   - 膈肌上移减少胸腔容积，降低肺活量
   - 睡眠中呼吸中枢敏感性下降，仅在显著低氧时才觉醒
3. **识别特殊表现**：若同时出现哮鸣音，应考虑“心源性哮喘”诊断。
4. **进行关键鉴别**：必须与阻塞性睡眠呼吸暂停综合征、支气管哮喘等非心源性疾病鉴别，重点依据：
   - 心脏病史及体征（如奔马律、颈静脉怒张）
   - 对利尿剂或强心药物的治疗反应

## 输出提示
该症状是左心衰竭的重要警示信号，提示肺淤血严重，需紧急评估并干预。
