# 休克衰竭期多器官功能障碍机制分析

## 适用场景
- 患者已确诊为休克且进入衰竭期
- 出现顽固性低血压、DIC 或意识障碍等终末期表现
- 需评估多器官（心、脑、肺、肝、肾）损伤风险

## 执行步骤
1. **确认微循环“无复流现象”**：即使大血管血流恢复，微血管仍因内皮肿胀、白细胞嵌塞或微血栓而无法再灌注。
2. **识别微血栓广泛形成**：在DIC背景下，凝血系统激活导致微循环阻塞。
3. **评估器官低灌流状态**：检查心、脑、肺、肝、肾等重要器官是否持续缺血。
4. **监测内环境破坏指标**：包括溶酶体酶释放、炎性细胞因子（如TNF-α、IL-1）升高、活性氧（ROS）爆发。
5. **判断细胞损伤程度**：观察线粒体功能障碍、ATP合成减少、细胞膜通透性改变等表现。
6. **关联临床症状**：
   - 呼吸困难 → 肺损伤
   - 少尿/无尿 → 肾损伤
   - 意识模糊或昏迷 → 脑损伤
   - 黄疸或转氨酶升高 → 肝损伤
7. **发出预警**：若符合上述机制，立即提示“患者已进入休克终末期，需启动器官支持与抗炎干预”。

## 注意事项
- 仅适用于休克衰竭期，不适用于代偿期或非休克状态
- 需结合实验室与影像学证据综合判断
- 本技能输出为风险预警，不替代具体治疗决策
