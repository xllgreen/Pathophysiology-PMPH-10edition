# 细胞周期内在调控异常与肿瘤发生的关系分析

## 适用对象
- 人类肿瘤细胞
- 癌前病变组织

## 使用前提
- 样本中存在基因突变、扩增、甲基化或染色体易位等遗传改变
- 需结合外源信号异常共同作用（本技能仅聚焦内在调控异常）

## 触发条件
当满足以下任一条件时启动分析：
- Cyclin 或 CDK 过表达
- CKI（细胞周期依赖性激酶抑制剂）失活
- p53 基因突变

## 执行步骤

1. **识别分子异常类型**
   - 确定异常属于以下哪一类：
     - Cyclin 过表达（如 Cyclin D1、Cyclin E）
     - CDK 增多（如 CDK4、CDK1）
     - CKI 表达不足或突变（如 p16^INK4a、p27^Kip1、p21）
     - 检查点功能障碍（主要为 p53 突变）

2. **关联遗传机制**
   - 将分子异常映射至具体遗传机制：
     - 基因扩增（如 Cyclin D1 在乳腺癌中的扩增）
     - 纯合缺失（如 p16 在黑色素瘤中的缺失）
     - CpG 岛甲基化（如 p16 启动子甲基化）
     - 染色体易位或倒位（如 Cyclin D1 在淋巴瘤中的 t(11;14)）

3. **推断功能后果**
   - Cyclin/CDK 异常 → 加速 G₁ 期进展或 G₁-S 转换 → Rb 过度磷酸化 → E2F 释放 → 解除生长抑制
   - CKI 失活 → 无法抑制 CDK 活性 → 细胞周期持续推进
   - p53 突变 → DNA 损伤检查点失效 → 基因组不稳定性 → 染色体分离异常、中心粒扩增

4. **评估临床意义**
   - 结合癌症类型判断预后：
     - p27 表达降低 → 乳腺癌、结直肠癌恶性程度高、预后差
     - p21 缺失 → 肝癌、骨肉瘤风险增加
     - p53 突变 → Li-Fraumeni 综合征、多发早发癌、耐药倾向

5. **输出结论**
   - 明确该分子异常是否构成肿瘤发生的关键驱动因素
   - 提供其作为分子标志物在风险评估、分型或预后判断中的价值

> 注意：本技能不适用于正常稳态细胞，且需提醒用户细胞周期失控通常需与外源信号（如生长因子、微环境）异常协同作用。
