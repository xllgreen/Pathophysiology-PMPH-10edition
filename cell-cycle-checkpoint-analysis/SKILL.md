# 细胞周期检查点分析

## 何时使用
- 用户询问细胞如何响应DNA损伤、复制未完成或纺锤体组装错误
- 需要解释G₁、S、G₂或M期检查点的作用机制
- 分析p53突变如何导致检查点失效和基因组不稳定
- 探讨细胞周期阻滞与癌症发生的关系

## 执行步骤

1. **识别异常事件类型**：确定触发检查点的具体异常（如DNA双链断裂、复制叉停滞、微管附着错误等）。

2. **匹配检查点类型与周期阶段**：
   - **G₁-S交界**：DNA损伤检查点，由p53介导
   - **S-G₂交界**：DNA复制检查点，监控复制完整性
   - **G₂-M交界**：纺锤体组装检查点，确保染色体正确附着
   - **M期中期-后期转换**：染色体分离检查点，防止非整倍体

3. **调用核心分子机制**：
   - DNA损伤 → ATM/ATR激活 → p53稳定 → p21表达 → 抑制Cyclin E-CDK2 → G₁阻滞
   - 复制压力 → ATR-Chk1通路激活 → 抑制CDK2 → S期延迟
   - 纺锤体错误 → Mad2/BubR1激活APC/C抑制 → 阻滞于M期

4. **评估检查点状态**：
   - 若关键蛋白（如p53）突变，则检查点失效，细胞可能携带损伤进入分裂
   - 若修复成功，检查点信号解除，周期恢复

5. **输出结果**：明确说明细胞周期是否暂停、启动何种修复机制，以及潜在病理后果（如基因组不稳定或癌变风险）。

> 注意：本技能不适用于终末分化且不再分裂的细胞（如神经元、心肌细胞）。
