# 乙酰胆碱系统异常与学习记忆障碍关联评估

## 何时使用
- 患者表现为进行性或急性学习记忆障碍；
- 已知或疑似存在海马、基底前脑、颞叶皮质等关键脑区的胆碱能神经元损伤；
- 可获取脑组织、脑脊液或影像学生物标志物数据以评估乙酰胆碱（ACh）或胆碱乙酰转移酶（ChAT）水平。

## 执行步骤
1. 确认是否存在胆碱能系统损伤的证据，例如：
   - 脑脊液中ACh浓度降低；
   - 脑组织活检或动物模型中ChAT活性下降；
   - 影像学显示基底前脑萎缩。
2. 判断ACh/ChAT下降是否与认知障碍在时空上一致：
   - 阿尔茨海默病：皮质和海马ChAT减少50%~90%；
   - 血管性痴呆：脑脊液ACh下降程度与痴呆评分正相关；
   - 脑震荡或反复缺血：基底前脑或海马胆碱能神经元显著减少。
3. 若确认胆碱能系统功能低下是主要机制，则可将ACh/ChAT水平作为评估认知障碍严重程度及治疗反应的生物标志物。
4. 注意排除非胆碱能主导的认知障碍（如额颞叶痴呆、路易体痴呆早期以外的类型）。

> **提示**：本技能聚焦于胆碱能通路，不适用于多巴胺、谷氨酸等其他神经递质系统主导的认知障碍。
