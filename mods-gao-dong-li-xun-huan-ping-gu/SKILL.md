# MODS高动力循环的血流动力学评估

## 适用条件
- 患者处于MODS早中期或晚期阶段
- 存在严重感染或全身炎症反应综合征（SIRS）
- 炎症介质大量释放
- **不适用于**心源性休克或单纯低血容量状态

## 执行步骤

### 1. 判断循环类型
- **早中期（高排低阻型）**：
  - 心脏指数 > 4.0 L/(min·m²)，可达10 L/(min·m²) 或更高
  - 外周血管阻力显著降低
  - 射血分数低于正常，提示心肌抑制
  - 主要由心率增快代偿，但心功能实际受损
- **晚期（低排低阻型）**：
  - 心脏指数 < 2.0 L/(min·m²)
  - 外周血管阻力仍低
  - 心排血量下降，提示心力衰竭

### 2. 分析低阻机制
- 炎性扩血管物质（如NO、PGI₂）大量释放
- 肝功能受损导致内源性扩血管物质灭活减少
- 芳香族氨基酸潴留干扰神经血管调节

### 3. 识别心肌抑制因素
- 高代谢状态加重心脏负担
- 急性肺损伤致低氧血症 → 肺动脉高压 → 右心负荷增加
- 炎症介质（TNF-α、IL-1）直接抑制心肌收缩
- 内毒素损伤心肌细胞
- DIC导致心肌微血栓与局灶坏死

### 4. 应用临床警示指标
- 平均动脉压 < 60 mmHg
- 心脏指数 < 2 L/(min·m²)
- 对正性肌力药物无反应

## 输出目标
通过血流动力学监测数据，明确MODS患者当前循环类型，为血管活性药物（如去甲肾上腺素）与强心药（如多巴酚丁胺）的选择提供依据。
