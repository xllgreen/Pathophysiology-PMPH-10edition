# 少尿型急性肾衰竭四阶段病程评估

## 何时使用
- 确诊为少尿型急性肾衰竭（尿量<400ml/日）
- 需动态评估病情进展与预后

## 执行步骤
1. **少尿期（1~2周）**：
   - 尿量<400ml/日
   - 低比重尿（1.010~1.015）、尿钠>40mmol/L
   - 高钾、酸中毒、氮质血症、水中毒
2. **移行期**：
   - 尿量>400ml/日
   - 肾小管开始修复，但内环境紊乱仍存
3. **多尿期（1~2周）**：
   - 尿量≥3000ml/日
   - 早期氮质血症仍高，后期易脱水、低钾
4. **恢复期**：
   - 尿量正常，肌酐/BUN正常
   - 功能完全恢复需数月
5. **输出阶段判断**：根据当前尿量及指标确定阶段，指导治疗强度与预后评估。
