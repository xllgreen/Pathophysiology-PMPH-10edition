# 急性期反应蛋白分析

## 适用条件
- 存在感染、烧伤、创伤、大手术等应激原
- 单核巨噬细胞已被激活并释放IL-1、TNF-α、IL-6
- 血浆中蛋白浓度出现急性变化（如CRP急剧上升、白蛋白下降）

## 执行步骤

1. **确认APP来源与定义**
   - 判断是否为急性期反应：血浆中特定蛋白浓度在应激后数小时内发生显著变化
   - 主要合成细胞为肝细胞，次要来源包括单核巨噬细胞、血管内皮细胞和成纤维细胞

2. **识别诱导机制**
   - 检测关键炎性细胞因子：IL-1、TNF-α、IL-6
   - 区分诱导模式：
     - IL-1/TNF-α 主要诱导 CRP、血清淀粉样蛋白A、补体C3
     - IL-6 主要诱导 α₁-抗胰蛋白酶、纤维蛋白原、铜蓝蛋白

3. **按变化幅度分类APP**
   - 剧烈升高（>1000倍）：CRP、血清淀粉样蛋白A（6~10小时达峰）
   - 中度升高（2~3倍）：α₁-酸性糖蛋白、α₁-抗糜蛋白酶、结合珠蛋白、纤维蛋白原（10~24小时）
   - 轻度升高（约50%）：铜蓝蛋白、补体C3（48~72小时）
   - 下降蛋白：白蛋白、转铁蛋白

4. **解读生物学功能**
   - 抗感染：CRP结合细菌细胞壁激活补体；纤维连接蛋白增强吞噬
   - 抗损伤：α₁-抗胰蛋白酶抑制蛋白酶；铜蓝蛋白清除氧自由基
   - 调节凝血与纤溶：早期促凝（纤维蛋白原↑），后期促溶（纤溶酶原↑）
   - 结合运输：结合游离Cu²⁺、血红素，防止毒性

5. **临床应用判断**
   - 通过检测特定APP（如CRP）水平评估炎症强度、疾病活动性及治疗反应

> 注意：本技能不适用于无炎症或非应激状态；部分APP（如白蛋白）在急性期反应中浓度下降，需综合判断。
