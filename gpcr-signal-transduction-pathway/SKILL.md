# GPCR介导的跨膜信号转导通路

## 何时使用
- 分析β肾上腺素、组胺、阿片类物质等配体如何通过GPCR影响细胞功能
- 解释cAMP/PKA、IP3/Ca²⁺/PKC等第二信使系统的激活机制
- 判断特定G蛋白亚型（Gs、Gi、Gq）对应的下游效应
- 排除非GPCR介导的信号通路（如RTK通路）

## 执行步骤
1. **确认配体与GPCR结合**：验证是否存在特异性配体（如激素、神经递质、光、气味）与7次跨膜GPCR结合，诱导构象变化。
2. **激活G蛋白**：GPCR激活膜内侧异源三聚体G蛋白（αβγ），促使GDP被GTP替换，Gα亚基解离。
3. **根据Gα类型启动下游通路**：
   - **Gs通路**：Gαs激活腺苷酸环化酶（AC）→ 生成cAMP → 激活PKA → 磷酸化靶蛋白（如L型Ca²⁺通道）
   - **Gi通路**：Gαi抑制AC → 降低cAMP水平 → 抑制PKA活性
   - **Gq通路**：Gαq激活磷脂酶C（PLC）→ 水解PIP₂生成IP₃和DAG → 
     • IP₃释放内质网Ca²⁺ → 激活CaMK
     • DAG与Ca²⁺协同激活PKC
4. **评估其他效应**：考虑G蛋白对PLA₂、PLD、MAPK、NF-κB、PI3K及离子通道的调控作用。
5. **输出结果**：明确最终生物学效应（如细胞增殖、代谢调节、器官功能改变）。

> 注意：仅适用于表达功能性GPCR的细胞；不适用于无GPCR或非7次跨膜受体介导的信号。
