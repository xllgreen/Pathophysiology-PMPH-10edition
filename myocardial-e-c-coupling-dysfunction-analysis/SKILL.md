# 心肌兴奋-收缩耦联障碍机制分析

## 适用场景
- 患者存在心肌缺血、缺氧、酸中毒或长期负荷过重
- 观察到心肌收缩力减弱或舒张延缓
- 需要从钙稳态角度解释心力衰竭的细胞机制

## 执行步骤

### 1. 判断是否存在胞外Ca²⁺内流障碍
- 检查是否存在以下情况：
  - 心肌组织去甲肾上腺素含量下降
  - β-肾上腺素受体密度减少或敏感性降低
  - 高钾血症（K⁺与Ca²⁺竞争抑制内流）
- 若存在，则胞质Ca²⁺浓度无法达到10⁻⁵ mol/L的收缩阈值

### 2. 评估肌质网钙转运功能
- 检查肌质网钙释放蛋白是否活性或含量降低 → Ca²⁺释放减少
- 检查肌质网钙泵是否活性或含量降低 → Ca²⁺摄取减少
- 双重后果判断：
  - 舒张期胞质Ca²⁺清除延迟 → 舒张功能受损
  - 收缩期肌质网Ca²⁺储备不足 → 收缩力下降

### 3. 分析肌钙蛋白与Ca²⁺结合能力
- 若存在酸中毒，H⁺与肌钙蛋白亲和力高于Ca²⁺，占据结合位点
- 即使胞质Ca²⁺达阈值，也无法有效触发收缩
- 同时注意酸中毒引发的次级效应：高钾血症（减少Ca²⁺内流）和肌质网钙结合蛋白亲和力异常

### 4. 对照正常耦联过程（用于机制对比）
- 兴奋期：L型钙通道开放 → Ca²⁺内流 → 触发肌质网释放 → [Ca²⁺]↑至10⁻⁵ mol/L → 与肌钙蛋白C结合 → 收缩
- 复极期：钙泵回收Ca²⁺ + 钠-钙交换排出 → [Ca²⁺]↓ → 舒张

## 输出建议
识别具体障碍环节，指导临床干预：
- 酸中毒 → 纠正pH
- 高钾血症 → 调整血钾
- 供血不足 → 改善心肌灌注
- 受体下调 → 考虑β受体调节治疗
