# 急性肾损伤（AKI）诊断与分级

## 何时使用
当成人患者出现以下任一情况时启动本技能：
- 血清肌酐在48小时内绝对值升高 ≥ 0.3 mg/dl（≥ 26.5 μmol/L）
- 或7天内血清肌酐增至 ≥ 1.5倍已知或推定的基础值
- 或尿量 < 0.5 ml/(kg·h) 持续 > 6小时

**注意**：不适用于终末期肾病已接受透析的患者（除非需评估残余肾功能）。

## 执行步骤
1. **确认前提条件**：
   - 获取近期（≤3个月）稳定状态下的基础血清肌酐值；若无，使用MDRD公式估算
   - 准确记录患者体重及连续尿量

2. **判断AKI是否成立**：满足上述任一诊断条件即诊断为AKI

3. **进行AKI分级**（取血清肌酐和尿量两个维度中的较高级别）：
   - **I级**：
     - 肌酐：升高 ≥ 26.5 μmol/L 或 达基础值1.5~1.9倍
     - 尿量：<0.5 ml/(kg·h)，持续6~12小时
   - **II级**：
     - 肌酐：达基础值2.0~2.9倍
     - 尿量：<0.5 ml/(kg·h)，持续 ≥ 12小时
   - **III级**：
     - 肌酐：达基础值3.0倍 或 绝对值升高 ≥ 353.6 μmol/L（4.0 mg/dl） 或 启动肾脏替代治疗
     - 尿量：<0.3 ml/(kg·h) 持续 ≥ 24小时 或 无尿 ≥ 12小时

4. **输出结果**：明确返回“是/否”诊断结论及具体分级（I/II/III），用于指导临床干预强度和预后评估。
