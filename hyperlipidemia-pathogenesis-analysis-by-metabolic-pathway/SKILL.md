# 高脂血症按代谢环节的发病机制分析

## 适用条件
- 患者确诊为原发性或继发性高脂血症
- 已明确血脂异常类型（高总胆固醇、高甘油三酯或混合型）
- 已排除急性应激状态及单纯饮食引起的短期波动

## 执行步骤

### 1. 评估外源性脂质摄取是否增加
- 检查是否存在高脂或高饱和脂肪酸饮食
- 评估肠道胆固醇吸收相关基因（如NPC1L1、ABCG5/ABCG8）是否异常
- 判断是否出现乳糜微粒（CM）合成增加或植物固醇血症表现

### 2. 评估内源性脂质合成是否亢进
- 分析肝脏胆固醇合成通路：HMG-CoA还原酶活性是否升高
- 检查激素水平（胰岛素、甲状腺激素升高；胰高血糖素、皮质醇降低）对合成的影响
- 评估是否存在胰岛素抵抗或肥胖导致的游离脂肪酸释放增加，进而促进VLDL合成

### 3. 评估脂质转运与分解代谢是否障碍
- **CM/VLDL代谢**：检查LPL活性、ApoCII缺乏、ApoCIII升高或ApoE2/E2基因型
- **LDL代谢**：筛查LDLR突变、ApoB100突变（如Arg3500Glu）、CETP活性变化
- **HDL逆转运**：排查ABCA1突变（Tangier病）、LCAT缺乏或CETP缺陷

### 4. 综合判断
- 区分原发性（基因突变主导）与继发性（糖尿病、甲减、肾病综合征等疾病或药物诱发）
- 确定主导异常环节（摄取↑、合成↑ 或 分解↓）
- 输出靶向干预建议方向（如饮食调整、酶替代、受体调节或病因治疗）

> 注意：多个机制可同时存在，需结合临床、生化及基因信息综合分析。
